# Campus-Budget-Tracker-Project-
This is the Official Group VI submission for the Campus Budget Tracker Project for COS 102
Student Budget Tracker

Project Description

Student Budget Tracker is a desktop application developed using Python to help students manage their finances effectively. The application allows users to record income and expenses, monitor their current balance, and generate financial reports.

The project was created to provide a simple and user-friendly budgeting tool for students who want to track their spending habits and improve financial management.

Features

- Add income transactions
- Add expense transactions
- Delete transactions
- Automatic balance calculation
- View transaction history
- Export financial reports to PDF
- Simple graphical user interface (GUI)

Tools Used

- Python 3.1.4
- Visual Studio Code (VS Code) 2023
- Tkinter (GUI Development)
- ReportLab (PDF Report Generation)

Installation

1. Clone the repository:

git clone 

2. Navigate to the project folder:

3. Install required libraries:

pip install reportlab

4. Run the application:

python main.py

How to Use

1. Launch the application.
2. Enter income or expense details.
3. Save transactions.
4. Monitor the updated balance.
5. Export reports to PDF when needed.
